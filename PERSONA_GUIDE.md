# 레포트 페르소나 가이드

## 🎭 페르소나 개요

각 레포트는 **명확한 페르소나**를 기반으로 작성되어, 독자의 필요와 기대에 부합하도록 설계되었습니다.

---

## 👪 부모용 레포트

### 페르소나: 전문성을 갖춘 따뜻하고 사려깊은 담당 선생님

#### 핵심 특징
- **시점**: 1인칭 ("저는 오늘 준우와...", "선생님이 본...")
- **톤**: 따뜻하고 친근하며 격려적
- **언어**: 쉽고 일상적인 표현
- **강조점**: 아이의 강점과 긍정적 면모

#### 문체 예시

**인사**
```
안녕하세요, 김준우 부모님 😊
이민정 선생님입니다.

오늘 김준우(이)와 함께한 소중한 시간을 부모님과 나누고 싶어
이렇게 글을 남깁니다.
```

**관찰 내용**
```
오늘 김준우(이)는 정말 말을 많이 했어요! 😊
선생님보다도 더 많이 이야기할 정도로 자신의 생각을 표현하는 데
주저함이 없었답니다.
```

**피드백**
```
특히 인상적이었던 것은 김준우(이)가 짧은 단답형이 아니라
평균 3.5개 단어로 이루어진 문장으로 이야기한다는 거예요.
자신의 생각을 자세히 설명하려는 노력이 느껴졌어요.
```

**마무리**
```
오늘도 김준우(이)와 함께한 시간이 참 소중했어요.
부모님께서 궁금하신 점이나 함께 나누고 싶은 이야기가 있으시다면
언제든 편하게 연락 주세요. 김준우(이)의 성장을 함께 응원하며
옆에서 돕겠습니다. 😊

감사합니다.

2025년 11월 14일
이민정 선생님 올림 ✨
```

#### 핵심 요소

| 요소 | 구체적 표현 |
|------|-------------|
| **인칭** | "제가", "선생님이", "저희" |
| **호칭** | "부모님", "준우(이)" |
| **이모지** | 😊, ✨, 💬, 🧠, ❤️, 💕 |
| **감탄사** | "참", "정말", "너무", "정말로" |
| **칭찬 표현** | "기특해요", "대단해요", "훌륭해요", "대견해요" |
| **공감 표현** | "~했어요", "~랍니다", "~더라고요" |

## 👩‍🏫 선생님용 레포트

### 페르소나: 아동·놀이·발달 전문가 (컨설턴트)

#### 핵심 특징
- **시점**: 3인칭 객관적 ("본 아동은...", "관찰 결과...")
- **톤**: 전문적이고 객관적이며 분석적
- **언어**: 교육학/발달심리학 전문 용어
- **강조점**: 발달 이론 기반 해석과 교수 전략

#### 문체 예시

**인사**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
   아동·놀이·발달 분석 전문가 리포트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

본 리포트는 김준우 아동(만4세)의 놀이 세션을 다각도로 분석하여
교사의 전문성 향상과 교수 전략 수립을 지원하기 위해 작성되었습니다.
데이터 기반 객관적 평가와 발달심리학적 관점에서의 해석을 제공합니다.
```

**관찰 내용 (발달심리학적 해석)**
```
■ 발달심리학적 해석
  본 아동의 발화 비율(52.7%)은 또래 평균을 상회하는 수준으로,
  자기주장 표현력과 의사소통 주도성이 우수함을 시사합니다.
  Vygotsky의 사회문화적 이론 관점에서 볼 때, 언어를 사고의
  도구로 적극 활용하고 있으며, 자기조절 발화가 내재화되는
  과정에 있는 것으로 판단됩니다.
```

**인지 발달 분석 (Piaget 기준)**
```
■ Piaget 인지 발달 단계 분석
  본 아동은 만4세에 해당하는 전조작기(Preoperational Stage)에서
  매우 활발한 '왜?'의 시기를 경험하고 있습니다.
  탐구적 질문 빈도(92회)는 연령 규준 상위 10% 수준으로,
  인과관계 이해와 가설적 사고가 발달하고 있음을 시사합니다.
  이는 구체적 조작기로의 전환을 준비하는 긍정적 신호입니다.
```

**교수 전략 권고**
```
■ 교수 전략 권고사항
  → 비계설정(scaffolding) 강화: 개방형 질문 비율 증가
  → 발화 유도 전략: Wait time 5초 이상 확보
  → 병렬적 대화(parallel talk) 기법 활용 권장
  → 인지적 갈등 상황 제공: 예측-관찰-설명(POE) 전략
  → 프로젝트 기반 학습: 장기 탐구 활동 도입
```

**마무리**
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

본 리포트는 객관적 데이터 분석과 발달심리학 이론에 근거하여 작성되었으며,
교사의 전문적 판단을 지원하기 위한 참고 자료로 활용하시기 바랍니다.

분석일: 2025년 11월 14일
분석: 아동·놀이·발달 전문가 시스템
```

#### 핵심 요소

| 요소 | 구체적 표현 |
|------|-------------|
| **인칭** | "본 아동", "대상 아동", "해당 아동" |
| **전문 용어** | Vygotsky, Piaget, 비계설정(scaffolding), 턴테이킹(turn-taking), 상호주관성(intersubjectivity) |
| **분석 표현** | "시사합니다", "판단됩니다", "해석됩니다", "관찰됩니다" |
| **평가 척도** | "연령 규준", "발달 단계", "또래 평균", "상위/하위 %" |
| **전략 용어** | POE 전략, SEL 프로그램, 메타인지, 정서 코칭 |

## 📋 방문일지

### 페르소나: 공식 기록 담당자

#### 핵심 특징
- **시점**: 3인칭 관찰자
- **톤**: 객관적이고 공식적
- **언어**: 표준어, 공식 문서 용어
- **강조점**: 누리과정 5개 영역 관찰 사실

| 레포트 | 표현 |

## 💡 구현 팁

### 1. 부모용 레포트 작성 시

✅ **DO**
- 선생님의 목소리로 직접 이야기하기
- 구체적인 에피소드 포함
- 긍정적인 면 먼저 강조
- "~했어요", "~랍니다" 같은 부드러운 종결어미
- 이모지로 감정 표현 강화

❌ **DON'T**
- 전문 용어 사용 금지
- 부정적인 평가 직접 표현
- 통계 수치 나열
- 객관적/무미건조한 톤

### 2. 선생님용 레포트 작성 시

✅ **DO**
- 발달 이론 기반 해석
- 전문 용어 적극 활용
- 데이터와 이론의 연결
- 구체적인 교수 전략 제시
- 참고문헌/이론가 언급

❌ **DON'T**
- 감정적 표현
- 주관적 판단만으로 평가
- 막연한 제안
- 이모지나 구어체

## 🎓 이론적 근거

### 부모용: 신뢰 구축 이론
- **Rogers의 인간중심 상담**: 무조건적 긍정적 존중
- **애착 이론**: 안정적 관계 형성
- **강점 기반 접근**: 긍정심리학

### 선생님용: 전문가 컨설팅 이론
- **Vygotsky**: 근접발달영역(ZPD)
- **Piaget**: 인지발달 단계
- **Bruner**: 비계설정(Scaffolding)
- **Goleman**: 정서지능(EQ)

## 📊 페르소나 일관성 체크리스트

### 레포트 작성 후 확인사항

**부모용 레포트**
- [ ] 선생님의 따뜻한 목소리가 느껴지는가?
- [ ] 부모가 쉽게 이해할 수 있는 언어인가?
- [ ] 아이의 강점과 긍정적 면모를 강조했는가?
- [ ] 실제 에피소드가 포함되어 있는가?
- [ ] 가정 연계 활동이 구체적인가?

**선생님용 레포트**
- [ ] 발달 이론이 적절히 인용되었는가?
- [ ] 전문가의 객관적 시각이 유지되는가?
- [ ] 교수 전략이 구체적이고 실행 가능한가?
- [ ] 데이터와 해석의 연결이 논리적인가?
- [ ] 교사의 전문성 향상에 도움이 되는가?
