# 놀이 세션 분석 프로젝트 요약

## 🎯 프로젝트 목표

선생님과 아동의 놀이 세션을 분석하여:
1. 아동의 언어/인지/사회정서 발달 지표 추출
2. 회사용, 부모용, 선생님용 맞춤 레포트 자동 생성
3. 여러 세션 간 비교 분석

---

## 📊 추출 가능한 지표 (현재 데이터 기준)

### ✅ 완전 구현 (높은 정확도)

| 지표 | 설명 | 데이터 소스 |
|------|------|------------|
| **아동발화비율** | 전체 대화에서 아동 발화가 차지하는 비율 | VTT (화자 구분) |
| **발화양** | 총 발화 횟수, 글자 수, 평균 길이 | VTT |
| **긍정/부정 비율** | 감정 키워드 빈도 분석 | VTT (텍스트 분석) |
| **주요 정서 단어** | 자주 사용된 감정 관련 단어 추출 | VTT |
| **주요 토픽** | 빈도 기반 관심사 키워드 | VTT |
| **대화 교대** | 턴 테이킹 패턴 분석 | VTT (화자 순서) |

### 🟡 부분 구현 (개선 여지)

| 지표 | 현재 방법 | 개선 방향 |
|------|----------|----------|
| **문제해결 발화** | 키워드 규칙 기반 ("왜", "어떻게") | LLM 분류 모델 적용 |
| **주제 지속도** | 세그먼트 간 키워드 중복도 | 의미론적 유사도 분석 |
| **맥락전환도** | 주제 지속도의 역수 | 대화 맥락 변화 감지 AI |

## 🏗️ 시스템 아키텍처

```
[입력: raw_data/세션폴더]
         ↓
[1단계: analyze_play_session.py]
   - VTT 파싱
   - 화자 분리
   - 지표 계산
         ↓
[출력: analysis_results/*.json]
         ↓
[2단계: generate_reports.py]
   - JSON 로드
   - 3가지 레포트 생성
         ↓
[출력: reports/*_report.txt]
```

### 배치 처리
```
[batch_analyze.py]
   - 모든 세션 자동 분석
   - 레포트 일괄 생성
   - 비교 분석 리포트
         ↓
[출력: comparison_report.txt + .csv]
```

## 📝 레포트 종류

### 1️⃣ 회사용 레포트 (Company Report)
**목적**: 정량적 데이터 확인, 시스템 검증

**내용**:
- 모든 지표의 수치와 통계
- 원 데이터 상세
- 종합 평가 알고리즘 결과

**활용**:
- 분석 시스템 검증
- 알고리즘 개선 근거
- 연구/개발 자료

---

### 2️⃣ 부모용 레포트 (Parent Report)
**목적**: 부모의 이해와 격려

**특징**:
- 쉬운 언어 (전문 용어 최소화)
- 긍정적 톤
- 아이의 강점 중심
- 구체적 예시 포함

**구성**:
```
💬 얼마나 많이 이야기했나요?
😊 어떤 감정을 표현했나요?
🎯 무엇에 관심이 있었나요?
🧠 문제해결 능력은 어떤가요?
🌟 종합 평가 (강점 중심)
💌 마무리 말씀
```

---

### 3️⃣ 선생님용 레포트 (Teacher Report)
**목적**: 교육적 개선과 실행 가능한 제안

**특징**:
- 발달 영역별 분석
- 구체적 교수 전략 제안
- 누리과정 연계
- 다음 세션 계획

**구성**:
```
📊 핵심 지표 요약
🎯 발달 영역별 분석
   - 언어 발달
   - 사회정서 발달
   - 인지 발달
🎲 놀이 패턴 분석
💡 교육적 제안 사항
📚 누리과정 연계
🔄 다음 세션 계획
✅ 교사 체크리스트
```

## 🔄 작업 프로세스

### 단일 세션 분석
```bash
# 1단계: 분석
python3 analyze_play_session.py "raw_data/[세션명]"

# 2단계: 레포트 생성
python3 generate_reports.py "analysis_results/[세션명]_analysis.json"
```

### 전체 세션 배치 처리
```bash
# 한 번에 모든 분석 + 레포트 + 비교
python3 batch_analyze.py
```
